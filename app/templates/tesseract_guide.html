Tesseract OCR Installation Guide

Learn how to install and configure Tesseract OCR to use the OCR functionality in Docuflex

Introduction

Tesseract OCR is an open-source optical character recognition engine that Docuflex uses to extract text from images and PDFs. This guide will walk you through the installation process for different operating systems.

Windows Installation

  1. Download the installer from the UB Mannheim GitHub repository: https://github.com/UB-Mannheim/tesseract/wiki
  2. Run the installer and follow the on-screen instructions
  3. Make sure to check the option "Add to PATH" during installation
  4. Verify installation by opening Command Prompt and typing:
tesseract --version

macOS Installation

The easiest way to install Tesseract on macOS is using Homebrew:

  1. Install Homebrew if you don't have it already:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. Install Tesseract OCR:
brew install tesseract
  1. Verify installation:
tesseract --version

Ubuntu/Debian Installation

Install Tesseract using apt:

sudo apt update
sudo apt install tesseract-ocr
sudo apt install libtesseract-dev

Verify installation:

tesseract --version

Installing Additional Languages

By default, Tesseract is installed with English language support. To install additional languages:

Windows

Download language data files from the Tesseract tessdata repository and place them in the Tesseract tessdata directory (usually C:\Program Files\Tesseract-OCR\tessdata).

macOS

brew install tesseract-lang

Ubuntu/Debian

sudo apt install tesseract-ocr-all

Or install specific languages:

sudo apt install tesseract-ocr-fra # French
sudo apt install tesseract-ocr-deu # German
sudo apt install tesseract-ocr-spa # Spanish

Troubleshooting

Common Issue: Tesseract Not Found

If you get an error like "tesseract is not recognized" or "Tesseract not installed", make sure:

  • Tesseract is properly installed
  • Tesseract is added to your system PATH
  • You have restarted your terminal or command prompt after installation
